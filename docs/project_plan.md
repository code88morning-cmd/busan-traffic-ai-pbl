# PBL Project Plan

## 프로젝트명
공공데이터 기반 부산시 교통량 예측 AI 웹서비스 개발

## 문제상황
부산은 관광지 이동, 항만 물류, 도심 출퇴근, 해안도로, 행사·축제, 기상 변화에 따라 교통량 변동이 크다. 학습자는 공공데이터를 활용하여 부산시 주요 권역의 교통량 또는 혼잡도를 예측하고, 시민과 행정 담당자가 이해하기 쉬운 웹서비스로 구현한다.

## 팀별 권장 주제
1. 해운대·광안리 관광 교통량 예측
2. 서면·부산역 도심 교통량 예측
3. 부산항 주변 물류 교통량 예측
4. 센텀시티·벡스코 행사일 교통량 예측
5. 기상 변화 기반 교통량 예측
