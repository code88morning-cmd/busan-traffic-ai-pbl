# Data Sources

본 프로젝트는 서울시 교통량 예측 실습 구조를 부산시 교통량 예측 PBL 프로젝트로 변경한 버전이다.

## 1. 교통량 데이터
- 1순위: 부산광역시 스마트교차로 접근로 교통량 정보
- 대안: 부산광역시 AVI 교통량 데이터

## 2. 기상 데이터
- 기상청 단기예보 또는 ASOS 관측자료
- 주요 변수: 기온, 강수량, 습도, 풍속

## 3. 대기질 데이터
- 부산광역시 대기질 정보 조회
- 주요 변수: PM10, PM2.5, O3, NO2, CO, SO2

## 4. 공휴일 데이터
- 한국천문연구원 특일 정보
- 파생변수: is_holiday

## 주의
API 인증키는 `.env` 또는 환경변수로 관리한다. 노트북이나 Python 파일에 인증키를 직접 작성하지 않는다.
